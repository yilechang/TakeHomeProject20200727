說明
------
### 安裝docker
官方文檔：[https://www.docker.com/products/docker-desktop](https://www.docker.com/products/docker-desktop)
### 啟動docker
```Bash
docker-compose up -d
```
### 前端頁面
[http://localhost:3000/](http://localhost:3000/)
------
### 關閉docker
```Bash
docker-compose down
```
